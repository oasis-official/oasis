프로젝트 생성

Maven 또는 Gradle 프로젝트를 만듭니다.

Eclipse + Maven

  • File→new→Other.. 선택

  • Maven→Maven Project 선택
  • Create a simple project에 체크 후 Next 버튼 클릭
  • group id, artifact id 등을 입력 후 Finish 버튼 클릭
  • resources에 services 디렉토리 추가

의존성 추가

pom.xml 파일에 oasis-core 의존성을 추가 합니다.

<dependency>
	<groupId>com.dongkuk</groupId>
	<artifactId>oasis-core</artifactId>
	<version>4.0.0</version>
</dependency>

Hello 비즈니스 로직 개발

간단한 메시지를 출력하는 메소드를 가진 클래스를 만듭니다.

Hello class 생성

  • src→main→java 에서 new→class 선택

  • package 와 name 을 적고 Finish 선택
  • 로직 작성
package oasis;

public class Hello {
	public void greeting(String name) {
		System.out.println(name + "! Hello.");
	}
}

모델 작성

모델 그리기

  • 다운 받은 모델러를 실행해서 BPMN diagram (Camunda Platform) 선택
  • 시작 이벤트의 이름을 변경하고 Task를 추가
  • 추가한 Task에 이름을 변경
  • Task 타입을 랜치 모양 아이콘을 눌러 Service Task 로 변경
  • End Event 를 추가하여 끝임을 알림

Service Task 설정

  • Java Class 를 실행하기 위해 설정을 변경하고 실행할 클래스 입력
  • 실행할 메소드 이름을 입력
  • resources/services 에 모델 저장

서비스를 실행할 클라이언트 만들기

  • main 메소드를 포함하는 Main class 만들기
  • my-first-oasis 서비스를 실행하는 main 메소드 작성
package oasis;

import java.util.HashMap;
import java.util.Map;

import com.dongkuk.oasis.TypedObject;
import com.dongkuk.oasis.context.DefaultServiceContext;
import com.dongkuk.oasis.context.ServiceContext;
import com.dongkuk.oasis.factories.NonTransactionalServiceStarterFactory;
import com.dongkuk.oasis.service.ServiceStarter;

public class Main {

	public static void main(String[] args) {
		ServiceStarter serviceStarter = 
				new NonTransactionalServiceStarterFactory().generateServiceStarter();

		// Build input values
		Map<String, TypedObject> input = new HashMap<>();
		input.put("name", new TypedObject("dongkuk"));

		// Make service context
		ServiceContext serviceContext = new DefaultServiceContext(input);

		serviceStarter.start("my-first-oasis", serviceContext);
	}
}
